# 🚀 AI Resume Analyzer

An AI-powered web application that analyzes resumes against job descriptions and provides intelligent feedback, skill gap analysis, and improvement suggestions.

## 🔥 Features

- 📄 Upload Resume (PDF)
- 📊 Match Score using NLP (TF-IDF + Cosine Similarity)
- 🧩 Skill Gap Analysis
- 🧠 AI-based Feedback
- 📈 Interactive Dashboard (Streamlit)

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLP (TF-IDF, Cosine Similarity)
- Plotly

## 🚀 How It Works

1. Upload resume
2. Paste job description
3. System analyzes similarity
4. Outputs:
   - Match Score
   - Matched Skills
   - Missing Skills
   - Suggestions
